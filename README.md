# Кто заходил

Статический VK Mini App, который автоматически показывает для текущего пользователя случайные выборки друзей и друзей друзей по трем вкладкам:

- за последний день
- за последнюю неделю
- за последний месяц

Важно: VK API не предоставляет список реальных посетителей страницы. Поэтому приложение отображает вероятный круг гостей на основе доступного графа друзей текущего пользователя.

## Что умеет

- получает профиль текущего пользователя через VK Bridge
- получает список друзей первого уровня через `friends.get`
- пытается собрать друзей второго уровня через публичные списки друзей ваших друзей
- разбивает вывод на табы `За день`, `За неделю`, `За месяц`
- не использует ручной ввод ID, а работает только для пользователя, который открыл приложение
- адаптировано под ПК и мобильные устройства

## Файлы

- `index.html` — разметка приложения
- `styles.css` — стили
- `app.js` — логика Bridge, VK API, табов и демо-режима

## Локальный запуск

Можно открыть `index.html` напрямую в браузере, но реальные данные будут доступны только внутри VK Mini Apps. Для локальной проверки верстки лучше поднять любой статический сервер:

```bash
python3 -m http.server 4173
```

После этого откройте `http://localhost:4173`.

## Для VK Mini Apps

1. Создайте приложение в кабинете VK Mini Apps.
2. Загрузите эти файлы на HTTPS-хостинг.
3. Убедитесь, что домен указан в настройках приложения.
4. Откройте мини-апп внутри VK: данные текущего пользователя и друзей загрузятся автоматически.

## Ограничения

- приложение не может узнать настоящих посетителей страницы
- у части пользователей списки друзей закрыты, поэтому друзей второго уровня может быть мало
- выборки в табах детерминированно перемешиваются по периоду, а не строятся по реальной истории посещений
